Casacore implementation notes

This page lists some of the implementation notes from casacore. If you want to install casacore, visit github.com/casacore/casacore.

Casacore was developed as part of the AIPS++ project. Some of the older notes mention AIPS++, you can probably substitute this with Casacore. We have tried to remove the really obsolete notes, if you're missing something please request it on the Issue tracker.
